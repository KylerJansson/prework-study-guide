Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • HTML attributes are always defined in teh beginning element, never the closing element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain 2 components: Property and Value.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: add command adds modifications in the current working branch to the staging area. -A indicates we want to add all changes.
  • git commit -m"description": adds message to describe what the commit contains.
  • git pull origin main: pulls down any changes from main branch to make sure we are synced.
  • git push origin "name of branch here": pushes changes to branch.
